# Kraken — VPN landing (макет)

Статический многостраничный сайт-макет для VPN-сервиса Kraken. Чистый HTML/CSS/JS, без сборки — готов к публикации на GitHub Pages.

## Структура

```
index.html      — главная (герой, «глубиномер», как это работает)
servers.html     — список серверов с поиском и фильтром по регионам
pricing.html     — тарифы с переключателем месяц/год
style.css        — общие стили и дизайн-токены
script.js        — интерактивность (поиск, фильтры, переключатель тарифов)
```

## Как выложить на GitHub Pages

1. Создай репозиторий и залей туда все файлы из этой папки.
2. В настройках репозитория → **Settings → Pages** выбери ветку `main` и папку `/root`.
3. Через пару минут сайт будет доступен по адресу `https://<твой-логин>.github.io/<репозиторий>/`.

## Локальный просмотр

Просто открой `index.html` в браузере — внешних зависимостей, кроме шрифтов Google Fonts, нет.

## Заметка

Это макет с демонстрационными данными (сервера, цены) — если захочешь превратить в реальный продукт, подключи бэкенд для аутентификации, оплаты и раздачи конфигов.
